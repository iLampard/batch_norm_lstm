# batch_norm_lstm


In an attempt to learn Tensorflow, I have implemented 
[Recurrent Batch Normalization](https://arxiv.org/abs/1603.09025) using Tensorflow 1.14.


## How to run

```python
python main.py 
```

## Requirement

```bash
tensorflow==1.14.0
numpy==1.16.4
```
